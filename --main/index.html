Quiet / Minimal / No Cards

把网站做得像一页排版，
不是一堆盒子。

这是一个从零开始做的小网站：也不靠花哨组件堆砌内容。它更像一本展开的册子，留白足够，节奏清楚，适合展示你的审美、文字、作品或者课程项目。

继续往下看
Why

一个更舒服的表达方式

有些内容只需要被安静地摆出来：标题、段落、图片、分隔线，然后让浏览者自然读下去。
01

连续排版

内容顺着视线往下走，不被一个个方块打断。适合叙述型、展示型、作品集型页面。

02

更少装饰

去掉多余的边框、阴影和强调色，让文字和节奏本身成为视觉中心。

03

更好修改

结构简单，HTML 和 CSS 都很直观。你后续替换成自己的内容会更方便。

Selected Notes
理念

风格先退后一步，内容才会向前一步

当页面不过度“设计”，用户反而更容易记住你真正想表达的东西。干净不等于空，克制也不等于无聊。

体验

手机端和桌面端都保持同一种气质

不是简单堆响应式，而是在不同屏幕上都保留留白、对齐和阅读节奏，让网站像一个完整作品。

适用

个人主页、作品展示、课程作业
